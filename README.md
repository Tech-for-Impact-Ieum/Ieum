# 이음

### Prerequisite
  - git
  - node.js, npm

### Local Setup
- Repository 다운 받기
  - `git clone https://github.com/Tech-for-Impact-Ieum/Ieum.git`

- 필요한 패키지 다운 받기
  - 다운 받은 Repository를 current dir로 두고 `cd Ieum`
  - 사용된 package 다운로드 `npm install`

- 실행하기
  - 터미널에서 `npm run dev`
  - 브라우저의 [http://localhost:3000](http://localhost:3000)에서 띄운 웹 확인
